# Portfolio

Portfolio pessoal estilo Vercel — dark, minimalista e tech.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Começar

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173).

## Personalizar

Edite `src/data/portfolio.ts` com seus dados:

- Nome, bio, links sociais
- Projetos e tags
- Skills e categorias

## Deploy

Pronto para deploy na [Vercel](https://vercel.com):

```bash
npm run build
```

Ou conecte o repositório GitHub diretamente na Vercel.
